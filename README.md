This lecture provides a high-level theoretical overview of neural networks, how they work, and how they are applied in modern AI systems.

### Topics Covered:

- AI vs Machine Learning vs Deep Learning  
- Biological inspiration of neural networks  
- Neuron computation:
  - Weighted sum  
  - Bias  
  - Activation functions  
- Network architecture:
  - Input layer  
  - Hidden layers  
  - Output layer  
- Forward propagation  
- Backpropagation and gradient descent (conceptual overview)  
- Epochs and batch training  
- Train / Validation / Test split  
- From shallow networks to deep learning  
- Real-world applications:
  - ChatGPT (Transformer Neural Networks)  
  - Self-driving cars (Convolutional Neural Networks)  
  - Fraud detection  
  - Medical diagnosis  

### Learning Objectives:

After this lecture, students should be able to:

- Explain how a neural network makes predictions  
- Describe how neural networks learn through optimization  
- Define key terms such as epoch, batch, loss, and generalization  
- Connect neural network theory to real-world AI systems  

This lecture is designed to be delivered in approximately **20 minutes** and focuses on conceptual clarity rather than implementation.
